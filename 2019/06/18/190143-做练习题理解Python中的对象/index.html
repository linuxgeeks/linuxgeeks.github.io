很多人都会误认为 list1=[10]，list3=['a']，因为他们以为每次 extendList 被调用时，列表参数的默认值都将被设置为 []，但实际上并不是这样，新的默认列表在函数被定义的那一刻只会创建一次。

当 extendList 被调用时，如果没有指定特定参数，默认的 list 的值随后将被使用。这是因为默认值参数会在函数被定义的时候被创建和计算，而不是在被调用的时候被创建和计算。因此 list1 和 list3 指向的是同一个对象，也就是默认列表。而 list2 是函数被调用时另外指定了一个对象，而不是使用默认的。
