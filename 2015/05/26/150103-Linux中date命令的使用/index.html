date

  • print or set the system date and time，打印或设置系统日期和时间
  • Rtc：real time clock，硬件时间
  • Ntp：Network time Protocol，网络时间协议

输出格式的控制

格式 FORMAT 控制着输出格式. 仅当选项指定为全球时间时本格式才有效

默认情况下,用 0 填充数据的空缺部分. GNU 的 date 命令能分辨在%和数字指示之间的以下修改.
- (连接号) 不进行填充
_ (下划线) 用空格进行填充
