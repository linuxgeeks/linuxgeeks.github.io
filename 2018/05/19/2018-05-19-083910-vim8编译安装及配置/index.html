安装与清理

打开 vim 并在正常模式下输入 :PluginList 即可看到插件的清单，执行 :PluginInstall 即可安装插件，安装完成后在最后一行会显示 Done

删除插件只需要在 vimrc 配置文件中注释掉插件，在 vim 中用 :PluginClean 进行清理
