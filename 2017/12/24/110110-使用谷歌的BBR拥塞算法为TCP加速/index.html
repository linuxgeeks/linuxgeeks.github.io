发表于 2017-12-24 更新于 2019-06-10 阅读次数：

BBR简述

Google 开源了其 TCP BBR 拥塞控制算法，并提交到了 Linux 内核，从 4.9 开始，Linux 内核已经用上了该算法。 BBR 算法的目的是要尽量跑满带宽，并且尽量不要有排队的情况。它可以起到单边加速 TCP 连接的效果。

TCP BBR 致力于解决两个问题

  • 在有一定丢包率的网络链路上充分利用带宽
  • 降低网络链路上的 buffer 占用率，从而降低延迟
